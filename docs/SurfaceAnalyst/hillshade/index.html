3D Hillshade Map

      • Theory
      • Basic steps

　　Hillshade is a raster map that simulates the shadow of the practical surface so as to show ups-and-downs of the terrain. By using an imaginary light source to shine the map and the aspect and slope information generated from the raster data, you can get the grayscale value of each cell. The hill facing the light has higher grayscale value, while the hill back to the light has lower gray value which is called the shaded area. This mountain shadow map generated from raster calculation has a very realistic 3D effect, so it is also called hillshade.

　　3D hillshade map has significant value in the description of 3D situation and analysis of terrain. Overlaying other information on a 3D hillshade map can enhance its application value and intuitive effect greatly.

　　Hillshade is mainly used for display. If you overlay the raster map with the hillshade map, and after that, set the transparency of the raster layer, then you can easily create a beautifully fine 3D geomorphological map. You need to constantly adjust the parameters for different application scenarios in order to achieve the best effect for the mapping such as the transparency and brightness of the raster layer. Then you can continue to add other layers on it such as the maps of land use, road and river so as to further enrich the map information.

Theory

　　To create a 3D hillshade map, you need to specify the position of unrealistic light through following two parameters.

  • Azimuth: Determines the direction of the light source, which is denoted by degree. As shown in the figure below, the azimuth is clockwise measured with the due north (0 degrees) as the reference direction, which is assigned the value from 0 degrees to 360 degrees, so the due north is also 360 degrees. The due east is 90 degrees, the due north is 180 degrees, and the due west is 270 degrees. The default value of the azimuth is 315 degrees.

  • Altitude Angle: The angle of inclination of the light source ranges from 0 degrees to 90 degrees. As shown in the figure below, when the altitude angle is 90 degrees, the light source vertically shines the earth surface. The default value of the altitude angle is 45 degrees.

Basic steps

  1. In the toolbox, click “Raster Analysis” > “Surface Analysis” > “Terrain Calculate” > “3D Hillshade Map” to open the “3D Hillshade Map” dialog box.
  2. Specify your raster dataset at the Source Dataset area to create a 3D hillshade map.
  3. Azimuth: Denoted by a positive degree within [0,360]. It is clockwise measured with the north as the reference direction.
  4. Altitude Angle: The altitude angle above the horizon. It is denoted by a positive degree, 0 degree denotes the horizon, and 90 degree denotes the positive direction above the head.
  5. Shadow Mode: There are three modes supported, including Illumination and Shadow, Shadow, and Illumination.
    • Illumination and Shadow: Takes both the azimuth and shadow into consideration.
    • Shadow: Only takes whether the current area is within the shadow in consideration.
    • Illumination: Only considers the azimuth of current position.

  6. Z Factor: When the unit at Z direction is different from it at X, Y direction, you can adjust the unit at Z direction with the Z Factor.

    • If their units are the same, the Z Factor is 1.
    • If their units are different, an appropriate factor is required. For example, the unit at X, Y direction is meter, but feet at Z direction, so the Z Factor should be 0.3048 because of 1 feet is 0.3408 meter.

  7. Set a name for the result dataset and specify a datasource to save it. Click “Run” image button to perform the operation. As following pictures show, the left one is original raster data and the middle one is a 3D hillshade map. The right one is obtained by overlaying the hillshade map with a raster layer (transparency is 40%).
