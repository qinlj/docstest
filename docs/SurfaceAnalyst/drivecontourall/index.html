Extract All Isolines

      • Basic steps
      • Related topics

　　Extract all contour lines which satisfies a certain condition from a DEM or Grid dataset.

Basic steps

  1. In the toolbox, click “Raster Analysis” > “Surface Analysis” > “Extract Isolines” > “Extract All Isolines” to open the “Extract All Isolines” dialog box.
  2. On the settings of parameters in Parameter Settings area, please refer to Extract isolines and isoregions.

  3. Result Info: The result information will be shown automatically after parameters are set.

    • Max Cell Value: The maximum value in the source dataset. It can not be changed.
    • Min Cell Value: The minimum value in the source dataset. It can not be changed.
    • Max Isoline: The maximum value of isolines in the target dataset.
    • Min Isoline: The minimum value of isolines in the target dataset.
    • Count: The number of isolines.

  4. Click “Run” image button to perform the operation.
