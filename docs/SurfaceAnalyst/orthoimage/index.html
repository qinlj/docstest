Orthophoto Image

      • Basic steps
      • Related topics

　　The orthographic function is used to create the orthopthoto image by using the digital differential correction technology to get the reasonable sunshine intensity of the current point through the neighboring raster elevation.

　　The orthophoto image has prominent effect of expressing terrain situations and can facilitate knowing about features of mountainous terrain. Through the overlays of other information, various spatial analysis can be done like finding locations.

Basic steps

  1. In the toolbox, click “Raster Analysis” > “Surface Analysis” > “Terrain Calculate” > “Orthophoto Image” to open the “Orthophoto Image” dialog box.
  2. Specify your raster dataset to create an orthophoto image at the Source Dataset area.
  3. NoValue Color: The color of Novalue data in the result data.
  4. Color Scheme: Set a color table for the result data.
  5. Click “Run” image button to create an orthophoto image. As following pictures show: the left one is original data, and the right one is the generated orthophoto image.
