Extract Specified Isoregions

      • Basic steps
      • Related topics

　　Extract a certain amount of isoregions which have one or more special values. You can enter a special value directly, or the system will generate series of characteristic values according to the range and interval you set. Also, you can import specific values through importing the *.txt file where the specific values are saved in.

Basic steps

  1. In the toolbox, click “Raster Analysis” > “Surface Analysis” > “Extract Isolines” > “Specified Isoregions” to open the “Specified Isoregions” dialog box.

  2. Click button to open the “Batch Add” dialog box where you can set following parameters for extraction of isoregions.

    • Start Value: The start value for creating isoregions.
    • End Value: The maximum value for creating isoregions.
    • Interval: Check it to specify the difference in elevation between two adjacent isoregions and then how many isoregions will be calculated by system.
    • Classes: Specify how many isoregions in the target dataset when the Interval is not set.
    • Auto calculate the end value: After the Interval is set, check the check box, the end value of isoregions will be calculated again as the Start Value and Interval.

  3. Specify your raster dataset for extracting isoregions, and then the maximum value and minimum value of your data will be shown automatically.

    • Max Cell Value: The maximum value in the source dataset. It can not be changed.
    • Min Cell Value: The minimum value in the source dataset. It can not be changed.

  4. On the settings of parameters in Parameter Settings area, please refer to Extract isolines and isoregions.

  5. Click “Run” image button to perform the operation.
